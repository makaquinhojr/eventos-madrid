## ✨ Características

- 🗺️ **Mapa interactivo** con geolocalización de eventos
- 🔍 **Buscador inteligente** en tiempo real
- 📅 **Filtros avanzados** por fecha, tipo y precio
- 📊 **Panel de estadísticas** con contadores animados
- 🌙 **Modo oscuro** con persistencia en localStorage
- 📱 **Diseño responsive** mobile-first
- ⚡ **Rendimiento optimizado** con carga instantánea
- 🎨 **Interfaz clean** y minimalista

---

## 🛠️ Tecnologías utilizadas

### Frontend
- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con variables CSS
- **JavaScript (Vanilla)** - Lógica sin frameworks
- **Leaflet.js** - Mapas interactivos
- **OpenStreetMap** - Tiles de mapas gratuitos
- **Font Awesome** - Iconografía

### Datos
- **JSON** - Base de datos de eventos
- **Geolocalización** - Coordenadas GPS precisas

### Hosting
- **GitHub Pages** - Hosting gratuito
- **Git** - Control de versiones

---

## 📊 Datos del proyecto

- **50 eventos reales** verificados manualmente
- **5 categorías**: Conciertos, Fiestas, Mercados, Cultural, Gastronomía
- **Eventos gratuitos y de pago**
- **URLs oficiales** de cada evento
- **Actualización**: Manual semanal

🎯 Funcionalidades
Filtros

    Por fecha: Hoy, Este fin de semana, Próximos 7 días, Este mes
    Por tipo: Conciertos, Fiestas, Mercados, Cultural, Gastronomía
    Por precio: Gratis, De pago
    Búsqueda: Por nombre, lugar o descripción

Mapa

    Zoom y navegación fluida
    Iconos diferenciados por tipo de evento
    Popups con información completa
    Animaciones de hover

Estadísticas

    Contador de eventos por categoría
    Eventos gratuitos destacados
    Actualización en tiempo real según filtros
